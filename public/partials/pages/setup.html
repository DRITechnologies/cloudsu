Welcome to Concord

Add AWS account to setup components
  • concord will create the following components
    • DynamoDB Table - accounts and settings data
    • SQS Queue - used for communication between autoscaled instances and concord
    • SNS Topic - needed for caputuring scale group events
    • S3 Bucket - storage of large Cloud Formation templates
    • IAM User - DynanoDB access only
AWS accounts and sensitive data will be encrypted and stored in dynamodb
